## What works right now

- 👀 **Screenshot** — tap → interactive window picker → shutter → clipboard → ⌘V
- **B / I / U** — Bold, Italic, Underline sent to frontmost app via CGEvent (50ms delay, postToPid)
- 🥵 / 🌬️ **Fan button** — toggles Macs Fan Control between Automatic ↔ Full blast
  - Mechanism: `defaults write` to MFC's plist + SIGKILL + relaunch
  - No Accessibility permission needed
  - **User wrote this version themselves — do not touch FanAction.swift**

## Architecture

- `TouchStripAction` protocol — 1 file + 1 line in AppDelegate to add any new button
- `ButtonRegistry` — DFRFoundation private API, registers all buttons in Control Strip
- `TouchStripButtonItem` — generic NSCustomTouchBarItem wrapping any action
- `AppDelegate` — registers: ScreenshotAction, BoldAction, ItalicAction, UnderlineAction, FanAction
- `tsDebugLog()` → `/tmp/ts-debug.txt`

## Signing

- `TouchStrip Dev` self-signed cert in Keychain — stable identity, permissions survive rebuilds
- Makefile: tries `TouchStrip Dev` first, falls back to ad-hoc

## Permissions needed (all granted)

- Accessibility — CGEvent key injection for B/I/U
- Screen Recording — screencapture window mode
