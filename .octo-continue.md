## Full implementation spec (Develop ready)

### Layout
```
[96pt tokens] [140pt Allow Once] [140pt Always Allow] [140pt Reject]
```

### Detection — FINAL DECISION (debate verdict)
- **2s baseline CGWindowListCopyWindowInfo polling**
- **NSWorkspace.didActivateApplicationNotification** for Claude Desktop → burst to **0.5s for 10s**
- Stop entirely when Claude Desktop not running
- QuickWindow signature: `ownerPID == claudePID`, `layer == 8`, `width < 600`, `height < 300`

### Response mechanism
- `claude.activate(options: .activateIgnoringOtherApps)`
- CGEvent `leftMouseDown` + `leftMouseUp` via `postToPid` at calculated position
- Button positions within dialog bounds:
  - **Allow Once**: x + width × 0.25, y + height × 0.65
  - **Always Allow**: x + width × 0.55, y + height × 0.65
  - **Reject**: x + width × 0.82, y + height × 0.65
- Does NOT move system cursor
- No AppleScript, no AX

### State machine
```
IDLE   → Claude Desktop activates → BURST_POLLING (0.5s/10s)
BURST  → QuickWindow detected     → DIALOG_ACTIVE (buttons bright)
ACTIVE → button tapped            → click + back to BURST
ACTIVE → dialog disappears        → BURST or IDLE
BURST  → 10s elapsed              → SLOW_POLLING (2s)
any    → Claude Desktop quits     → IDLE (no polling)
```

### Button visual states
- **Idle**: dim (alpha 0.35), grey
- **Active**: Allow Once = green, Always Allow = blue, Reject = red
